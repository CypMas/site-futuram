# site-futuram

Déploiement GitHub Pages pour le dossier `site futuram`.

Instructions rapides :

1. Pousser ce dépôt sur GitHub (remplacez `URL_DU_DEPOT`).

```bash
git remote add origin URL_DU_DEPOT
git push -u origin main
```

2. Activer GitHub Pages : `Settings` → `Pages` → `Branch: main` + `Folder: /(root)` → `Save`.

Remarque : si vous voulez l'URL racine `https://VOTRENOM.github.io/`, nommez le dépôt `VOTRENOM.github.io`.
